Плагин "Пылесос" (версия 0.1.5) для LiveStreet 0.5 и 0.4х

Автор: Бишовец Николай (netlanc)
Связаться можно на сайте http://netlanc.net/ либо в личке на http://livestreet.ru/ (netlanc)

УСТАНОВКА

  1. Скопировать все файлы из архива в папку /plugins/ на Вашем сайте
  2. Войти на сайт под логином администратора
  3. Перейти по адресу http://<ваш_сайт>/admin/plugins
  4. Активировать плагин "Пылесос"
  5. "Пылесосить" перейдя по адресу http://<ваш_сайт>/settings/cleaner/
  6. При желании поставить в cron файл plugins/cleaner/cron.php ( например 0 * * * * /usr/bin/wget -O - -q http://_ваш_сайт_)/plugins/cleaner/cron.php)
  7. Радоваться жизни

УДАЛЕНИЕ ПЛАГИНА

  Удаление плагина происходит так же, как уделение остальных плагинов:
  1. Войти на сайт под логином администратора
  2. Перейти по адресу http://<ваш_сайт>/admin/plugins
  3. Деактивировать плагин "Пылесос"
  4. Отметить плагин "Пылесос" и нажать кнопку "Удалить плагины"
